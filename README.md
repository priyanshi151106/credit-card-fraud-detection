# 💳 Credit Card Fraud Detection using Ensemble Learning

This project presents a machine learning-based approach to detect fraudulent credit card transactions using ensemble learning techniques and data balancing methods.

---

## 🔍 Overview

* Developed fraud detection models using **Random Forest, XGBoost, and LightGBM**
* Addressed severe class imbalance using **SMOTE and SMOTE-Tomek**
* Focused on minimizing **false negatives** to reduce financial risk

---

## 📊 Key Results

* Achieved **ROC-AUC score of ~0.98**
* Reduced **false negatives from 33 → 13**
* Improved fraud detection recall while maintaining precision
